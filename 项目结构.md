## 核心文件说明

### sql_node_parser_v2.py (22K, 750行)
**功能**：SQL节点解析器 + SQL预处理器

**包含类**：
- `SQLPreprocessor` - 处理Oracle特殊语法和模板变量
- `SQLNodeParser` - 解析SQL，识别节点和关系
- `Node` - 节点数据类
- `Relationship` - 关系数据类

**核心功能**：
- 解析多种SQL方言（Oracle、MySQL、PostgreSQL等）
- 识别CTE、子查询、派生表、物理表、UNION等节点
- 建立CONTAINS和REFERENCES关系
- 自动预处理Oracle特殊语法（DECODE、TO_CHAR、NVL等）

### import_to_neo4j.py (11K, 381行)
**功能**：将SQL解析结果导入Neo4j图数据库

**包含类**：
- `Neo4jImporter` - Neo4j导入器

**核心功能**：
- 连接Neo4j数据库
- 创建节点标签（RootNode、CTE、Table、ScalarQuery等）
- 创建关系（CONTAINS、REFERENCES）
- 命令行参数处理

## 使用方式

### 命令行使用
```bash
# 解析SQL文件
python3 sql_node_parser_v2.py 原始.sql oracle

# 导入到Neo4j
python3 import_to_neo4j.py 原始.sql --dialect oracle
```

### Python代码使用
```python
from sql_node_parser_v2 import SQLNodeParser

parser = SQLNodeParser(sql_content, dialect='oracle')
nodes, relationships = parser.parse()
```
